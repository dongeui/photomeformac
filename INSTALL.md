# Trove for Mac 설치 가이드

이 문서는 처음 Trove를 설치하는 분을 위한 안내입니다. 개발자용 빌드/서명 절차는 [docs/mac/RELEASE_CHECKLIST.md](docs/mac/RELEASE_CHECKLIST.md)를 보세요.

## 준비물

- macOS 14 (Sonoma) 이상
- 정리할 사진이 들어 있는 폴더 (로컬 디스크, 외장하드, NAS, 또는 Apple 사진 보관함)

> Trove는 원본 사진을 **읽기만** 합니다. 사진을 옮기거나 수정하지 않습니다.

## 1. 다운로드

[Releases 페이지](https://github.com/dongeui/photomeformac/releases)에서 최신 `Trove.dmg`를 받습니다.

> 파일이 큽니다(약 500MB+). 이미지 AI 모델이 앱 안에 들어 있어서, 설치 후 인터넷 없이도 바로 검색·자동 태그가 동작합니다.

## 2. 설치

1. 받은 `Trove.dmg`를 더블클릭해서 엽니다.
2. 나타난 창에서 **Trove 아이콘을 옆의 `Applications` 폴더로 드래그**합니다.
3. DMG 창을 닫고, 바탕화면/Finder 사이드바의 디스크 이미지를 꺼냅니다(우클릭 → 추출).

> ⚠️ DMG 안에서 바로 실행하지 마세요. 반드시 **Applications로 옮긴 뒤** 실행해야 정상 동작합니다.

## 3. 첫 실행

1. `응용 프로그램`(Applications) 폴더에서 **Trove**를 더블클릭합니다.
2. "인터넷에서 다운로드된 앱"이라는 확인 창이 뜨면 **[열기]**를 누릅니다.
   - 만약 "확인되지 않은 개발자" 등으로 열리지 않으면, Trove 아이콘을 **우클릭 → [열기]** → 다시 **[열기]**를 누르세요. (첫 1회만 필요)
3. **첫 실행은 수십 초 정도 멈춘 듯 보일 수 있습니다.** macOS가 큰 앱을 처음 검사하기 때문이며 정상입니다. 잠시 기다리세요.

## 4. 사진 폴더 선택

1. 앱이 열리면 가운데 화면에서 **[사진 폴더 선택]**을 누릅니다.
   - 또는 Finder에서 폴더를 앱 창으로 **드래그&드롭**해도 됩니다.
   - Apple 사진 보관함이 감지되면 **[Apple Photos 추가]** 버튼이 보입니다.
2. macOS가 "Trove가 이 폴더에 접근하려 합니다" 같은 권한 창을 띄우면 **[허용]**을 누릅니다.
3. 폴더를 고르면 백엔드가 자동으로 시작되고 스캔이 시작됩니다.

## 5. 첫 분석 기다리기

- 첫 분석은 사진 수에 따라 **수십 분에서 수 시간**이 걸릴 수 있습니다.
- **창을 닫아도 됩니다.** 메뉴 막대(상단 바)의 Trove 아이콘이 계속 떠 있고, 백그라운드에서 진행됩니다.
- 진행 상황은 창 상단의 상태 표시와 Dock 배지에서 볼 수 있습니다.
- 작업이 끝나면 알림으로 알려줍니다. (첫 실행 시 알림 권한을 묻습니다 — 원치 않으면 거부해도 동작에는 지장 없습니다.)

## 자주 겪는 문제

**"포트 8000이 사용 중입니다"가 뜹니다.**
다른 프로그램이 8000 포트를 쓰고 있습니다. 그 프로그램을 종료하거나, 메뉴바에서 **[종료]** 후 앱을 다시 켜 보세요. 앱이 다시 켜질 때 자동으로 빈 포트를 찾아 백엔드를 띄웁니다.

**마운트 해제된 폴더가 있다고 나옵니다.**
NAS나 외장하드가 연결 해제된 상태입니다. 다시 연결하면 다음 자동 동기화에서 인식됩니다. 바로 반영하려면 메뉴바에서 **[종료]** 후 앱을 다시 켜세요. 연결이 끊겨도 앱이 종료되지 않고, 그 사진들은 썸네일로 계속 볼 수 있습니다.

**앱이 열리지 않습니다 ("확인되지 않은 개발자").**
3번 항목의 우클릭 → [열기] 방법을 쓰세요. 정식 서명/공증된 배포본에서는 보통 그냥 [열기]만으로 실행됩니다.

**다른 기기(폰 등)에서도 보고 싶습니다.**
Mac 앱은 본인 Mac에서만 접근하는 local-only로 동작합니다(LAN 공유 기능은 제거됐습니다). 여러 기기에서 공유가 필요하면 Docker/서버 배포를 사용하세요.

## 업데이트

새 버전이 나오면 앱이 자동으로 감지해 알려줍니다. **[지금 설치]**를 누르면 백그라운드에서 받아 교체하고 재시작합니다. (앱이 크기 때문에 다운로드에 시간이 걸릴 수 있습니다.)

## 데이터 위치

- 원본 사진: **건드리지 않습니다.** 사용자가 선택한 폴더 그대로 둡니다.
- 라이브러리 DB·썸네일·모델 캐시: 앱 데이터 폴더(`~/Library/Application Support/`)에 저장됩니다. 폴더를 다시 선택해도 사람 이름/별명/병합 결과는 보존됩니다.
